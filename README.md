# jhlsms
Sms package For JHL

Please ensure you add the following variables on settings

TWILIO_SID = 'Your SID '

TWILIO_TOKEN= 'Your Token '

How to use Package

Example

pip install twilio

pip install git+https://github.com/imadege/jhlsms

import sms

sms.send("twilio","+254724454978" "test message")
